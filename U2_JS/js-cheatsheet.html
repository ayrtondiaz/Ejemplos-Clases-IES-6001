JavaScript Cheatsheet

Métodos de Arrays & Objetos — referencia rápida

[ ] Arrays — Mutar
.push() muta

Agrega uno o más elementos al final.

const a = [1, 2];
a.push(3);        // [1, 2, 3]
.pop() muta

Elimina y devuelve el último elemento.

const a = [1, 2, 3];
a.pop();          // 3  →  a = [1, 2]
.unshift() muta

Agrega elementos al inicio.

const a = [2, 3];
a.unshift(1);     // [1, 2, 3]
.shift() muta

Elimina y devuelve el primer elemento.

const a = [1, 2, 3];
a.shift();        // 1  →  a = [2, 3]
.splice() muta

Agrega, elimina o reemplaza elementos en cualquier posición.

const a = ['a','b','c','d'];
a.splice(1, 2, 'x'); // ['a','x','d']
// desde idx 1, borra 2, inserta 'x'
.sort() muta

Ordena el array (por defecto como strings).

const a = [3, 1, 2];
a.sort((a, b) => a - b); // [1, 2, 3]
.reverse() muta

Invierte el orden del array.

const a = [1, 2, 3];
a.reverse();      // [3, 2, 1]
.fill() muta

Rellena todo (o parte) del array con un valor.

const a = [1, 2, 3];
a.fill(0);         // [0, 0, 0]
a.fill(9, 1, 2);   // [0, 9, 0]
[ ] Arrays — Nuevo Array
.map() nuevo arr

Transforma cada elemento y devuelve un nuevo array.

[1,2,3].map(x => x * 2);
// [2, 4, 6]
.filter() nuevo arr

Devuelve un nuevo array con los que cumplen la condición.

[1,2,3,4].filter(x => x > 2);
// [3, 4]
.flat() nuevo arr

Aplana arrays anidados al nivel indicado.

[[1,2],[3,[4]]].flat(1);
// [1, 2, 3, [4]]
[[1,2],[3,[4]]].flat(Infinity);
// [1, 2, 3, 4]
.flatMap() nuevo arr

map() + flat(1) en un solo paso.

[1,2,3].flatMap(x => [x, x*2]);
// [1, 2, 2, 4, 3, 6]
.slice() nuevo arr

Copia una porción del array (sin mutarlo).

['a','b','c','d'].slice(1, 3);
// ['b', 'c']
.concat() nuevo arr

Combina dos o más arrays.

[1].concat([2], [3]);
// [1, 2, 3]
[ ] Arrays — Buscar & Comprobar
.find() valor

Devuelve el primer elemento que cumple la condición.

[5,12,8].find(x => x > 10);
// 12
.findIndex() valor

Devuelve el índice del primer elemento que cumple.

[5,12,8].findIndex(x => x > 10);
// 1
.indexOf() valor

Índice de la primera aparición (o -1).

['a','b','c'].indexOf('b');
// 1
.includes() boolean

¿El array contiene este valor?

[1,2,3].includes(2);  // true
[1,2,3].includes(9);  // false
.some() boolean

¿Algún elemento cumple la condición?

[1,3,5].some(x => x % 2 === 0);
// false (ninguno es par)
.every() boolean

¿Todos los elementos cumplen?

[2,4,6].every(x => x % 2 === 0);
// true (todos pares)
[ ] Arrays — Reducir & Iterar
.reduce() valor

Acumula todos los elementos en un solo valor.

[10,20,30].reduce((acc, x) => acc + x, 0);
// 60
.forEach() void

Ejecuta una función por cada elemento (no devuelve nada).

['a','b'].forEach((v, i) => {
  console.log(i, v);
}); // 0 'a'  1 'b'
.join() string

Une los elementos en un string con el separador dado.

['hola','mundo'].join(' ');
// 'hola mundo'
Array.from() nuevo arr

Crea un array desde un iterable o con un mapeo.

Array.from({length: 3}, (_,i) => i);
// [0, 1, 2]
Array.from('abc');
// ['a', 'b', 'c']
{ } Objetos — Leer
Object.keys() array

Devuelve las claves (keys) del objeto.

const o = { a: 1, b: 2 };
Object.keys(o);   // ['a', 'b']
Object.values() array

Devuelve los valores del objeto.

const o = { a: 1, b: 2 };
Object.values(o);  // [1, 2]
Object.entries() array

Devuelve pares [clave, valor].

const o = { a: 1, b: 2 };
Object.entries(o);
// [['a', 1], ['b', 2]]
Object.hasOwn() boolean

¿El objeto tiene esa propiedad propia?

const o = { x: 10 };
Object.hasOwn(o, 'x'); // true
Object.hasOwn(o, 'y'); // false
{ } Objetos — Crear & Combinar
Object.assign() muta target

Copia propiedades de uno o más objetos al target.

const a = { x: 1 };
Object.assign(a, { y: 2 });
// a = { x: 1, y: 2 }
{ ...spread } nuevo obj

Crea un nuevo objeto combinando otros (inmutable).

const a = { x: 1 };
const b = { ...a, y: 2 };
// { x: 1, y: 2 }  (a no cambia)
Object.fromEntries() nuevo obj

Convierte pares [clave, valor] en un objeto.

const pares = [['a',1],['b',2]];
Object.fromEntries(pares);
// { a: 1, b: 2 }
Object.freeze() inmutable

Congela el objeto: no se puede modificar.

const o = Object.freeze({ a: 1 });
o.a = 99; // silenciosamente falla
// o.a sigue siendo 1
⚡ Combos Útiles
Destructuring sintaxis

Extraer valores de arrays y objetos.

const [a, , c] = [1, 2, 3]; // a=1, c=3
const {name, age} = {name:'Ana', age:25};
Optional Chaining sintaxis

Acceder a propiedades profundas sin error.

const user = { address: { city: 'BA' } };
user.address?.city;     // 'BA'
user.phone?.number;    // undefined
filter + map nuevo arr

Filtrar y luego transformar — combo clásico.

const precios = [100, 50, 200, 30];
precios
  .filter(p => p > 50)
  .map(p => p * 1.21);
// [121, 242] (con IVA)
entries + reduce valor

Iterar un objeto y acumular un resultado.

const cart = { pan:2, leche:3 };
Object.entries(cart)
  .reduce((s,[k,v]) => s + v, 0);
// 5 (total items)
